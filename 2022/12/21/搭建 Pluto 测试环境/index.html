搭建 Pluto 测试环境

0 介绍

Pluto 是一个独立的用户管理系统，它可以用来搭建一个测试环境，并且可以自动化地搭建一个简单的网站。 这个项目源自于简单日语团队的其中一位创始人: leeif。

1 环境需求

  1. Go Version，需要 >= 1.13, <= 1.16
  2. MySQL，5.7+
    1. Mac 系统的话，可以通过 brew 来安装
    2. 如果像我这样，用 brew 的方法安装有问题的话，可以下载 DMG 文件来安装，不过官方已经不提供下载了，我找到的是：http://ftp.iij.ad.jp/pub/db/mysql/Downloads/MySQL-5.7/mysql-5.7.25-macos10.14-x86_64.dmg

2. 安装和启动

  1. git clone git@github.com:MuShare/pluto.git
  2. 打开 pluto 目录，在项目根目录创建配置文件，config.yaml 或者 config.json，这边采用 yaml 格式
  1. 需要创建一些表格，pluto 提供了 migrate 的命令行
  1. 启动服务

服务器启动之后可以访问 http://localhost:8010/web/login 具体的 API 可以查看 pluto/docs/api.md 所有的 API 请求都是 JSON 格式。

  1. 新建 Application 如果没有新建 Application 的话，那么就无法进行后续的操作了
  1. 新建用户 Pluto 目前支持的用户注册方式是邮件，还可以绑定微信，谷歌和苹果账号，这边采用邮件注册。 在 config 里面配置了 skip_register_verify_mail，所以不需要验证邮箱。
  1. 登录 通过登录，我们可以获取 refresh_token 和 access_token。
  1. 删除用户 删除的时候我们需要 JWT，也就是登录的时候获得的 access_token，将这个 Token 放在请求头里面。

Reference

  2. https://reqbin.com/req/c-dwjszac0/curl-post-json-example
